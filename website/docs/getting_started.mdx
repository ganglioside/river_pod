[Riverpod]의 내부 메커니즘에 대해 알아보기 전에 기본부터 시작하겠습니다. [Riverpod]를 설치한 다음 "Hello world"를 작성합니다.

## 설치해야 할 패키지

무엇보다 먼저 [Riverpod]가 약간 다른 용도로 여러 패키지에 분산되어 있다는 사실을 알아야 합니다
설치하려는 [Riverpod]의 변형판은 만들고 있는 앱에 따라 다릅니다.

<!-- Before anything, you need to be aware that [Riverpod] is spread across multiple packages, with slightly different usage.  
The variant of [Riverpod] that you will want to install depends on the app you are making. -->

다음 표를 참고하여 사용할 패키지를 결정할 수 있습니다.

<!-- You can refer to the following table to help you decide which package to use: -->

| 앱 타입                  | 패키지명                                                                        | 설명                                                                   |
| ------------------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| Flutter + [flutter_hooks] | [hooks_riverpod]                                                                   | [flutter_hooks]와 [Riverpod]를 함께 사용할 수 있습니다.                    |
| Flutter 전용              | [flutter_riverpod]                                                                 | Flutter 애플리케이션에 [Riverpod]를 사용하는 기본 방법                      |
| Dart 전용 (Flutter 없음)    | [riverpod](https://github.com/rrousselGit/river_pod/tree/master/packages/riverpod) | Flutter와 관련된 모든 클래스가 제거된 [Riverpod] 버전입니다. |

## 패키지 설치하기

일단 설치하려는 패키지를 알게 되면, `pubspec.yaml`에 다음 코드를 추가하세요.

<!-- Once you know what package you want to install, proceed to add the following to your `pubspec.yaml`: -->

<Tabs
  groupId="riverpod"
  defaultValue="hooks_riverpod"
  values={[
    { label: 'Flutter + flutter_hooks', value: 'hooks_riverpod', },
    { label: 'Flutter', value: 'flutter_riverpod', },
    { label: 'Dart only', value: 'riverpod', },
  ]}
>
<TabItem value="hooks_riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter:
    sdk: flutter
  flutter_hooks: ^0.18.0
  hooks_riverpod: ^2.0.0-dev.0
```

그런 다음 `flutter pub get`을 실행합니다.
<!-- Then run `flutter pub get`. -->

</TabItem>
<TabItem value="flutter_riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.0.0-dev.0
```

그런 다음 `flutter pub get`을 실행합니다.
<!-- Then run `flutter pub get`. -->

</TabItem>
<TabItem value="riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0-0 <3.0.0"

dependencies:
  riverpod: ^2.0.0-dev.0
```
그런 다음 `dart pub get`을 실행합니다.
Then run `dart pub get`.

</TabItem>
</Tabs>

이것이 전부입니다. [Riverpod]를 추가했습니다.
<!-- That's it. You've added [Riverpod] to your app! -->
